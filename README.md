### Support
iOS 10.0 and later

### Install
1. Settings > General > Profile And ... > Trust app
2. Open app
3. Quit app, settings
4. Open Settings > General > Add new keyboard
5. Add LeKeyboard
6. Quit Settings and open keyboard from iMessage, Safari...
